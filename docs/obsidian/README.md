# WA CRM v2 Obsidian Vault

This directory is an Obsidian-compatible project memory vault.

Open `docs/obsidian/` as a vault in Obsidian if you want graph links, backlinks, and search. The repo copy is the canonical versioned memory record.

## Structure

- `index.md` - navigation index for active memory notes.
- `notes/` - dated project memory notes.
- `templates/` - reusable note templates.

## Rules

Follow `docs/OBSIDIAN_MEMORY_STANDARD.md` before adding or updating notes.
